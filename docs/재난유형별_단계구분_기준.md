# 재난 유형별 "단계" 구분 기준

> 정리 기준일: 2026-07-13

## 두 가지 서로 다른 단계 체계

한국의 재난 관련 "단계"는 실제로 **두 개의 독립된 체계**가 있고, 재난에 따라 하나만 있거나 둘 다 있습니다.

| 체계 | 근거 | 단계명 | 발령 주체 |
|---|---|---|---|
| **① 기상특보** | 기상법 제13조, 예보업무규정(기상청훈령) | 주의보 → 경보 (→ 중대경보, 폭염만 해당) | 기상청장 |
| **② 위기경보** | 「재난 및 안전관리 기본법」 제38조 | 관심 → 주의 → 경계 → 심각 | 각 재난관리주관기관의 장(대부분 중앙부처 장관, 국가적 재난은 행정안전부장관) |

**위기경보의 법적 원칙**(제38조제2항): *"재난 피해의 전개 속도, 확대 가능성 등 재난상황의 심각성을 종합적으로 고려하여 관심·주의·경계·심각으로 구분"*하되, **다른 법령에서 별도 기준을 정한 경우 그 기준을 따릅니다.** 즉 아래처럼 하위법령(시행령)이나 부처 훈령에서 재난별로 구체적인 수치 기준을 별도로 정해둔 경우가 많고, 그런 기준이 없는 재난은 **재난관리주관기관이 사고 규모·인명피해·확산 가능성을 그때그때 종합 판단**해서 발령합니다(정량 기준 비공개).

일반적 의미는 다음과 같습니다.

| 단계 | 의미 |
|---|---|
| 관심 | 위기징후 관련 현상이 나타나지만 활동 수준이 낮아 위기로 발전할 가능성이 적은 상태 |
| 주의 | 위기징후 활동이 비교적 활발하여 위기로 발전할 수 있는 경향성이 나타나는 상태 |
| 경계 | 위기징후 활동이 매우 활발하여 위기로 발전할 가능성이 농후한 상태 |
| 심각 | 위기징후 활동이 매우 활발하여 위기 발생이 확실시되는 상태 |

# 1부. 자연재난 (24종)

## 강풍 (기상특보만 있음)
| 단계 | 기준 |
|---|---|
| 주의보 | 육상 풍속 14m/s(50.4km/h) 이상 또는 순간풍속 20m/s(72km/h) 이상 (산지 17m/s·25m/s) |
| 경보 | 육상 풍속 21m/s(75.6km/h) 이상 또는 순간풍속 26m/s(93.6km/h) 이상 (산지 24m/s·30m/s) |
*별도 위기경보 없음(태풍에 의한 강풍은 「풍수해 위기경보」에 포함). 출처: 기상청 예보업무규정*

## 풍랑 (기상특보만 있음)
| 단계 | 기준 |
|---|---|
| 주의보 | 해상 풍속 14m/s 이상이 3시간 이상 지속 또는 유의파고 3m 이상 |
| 경보 | 해상 풍속 21m/s 이상이 3시간 이상 지속 또는 유의파고 5m 이상 |
*출처: 기상청 예보업무규정*

## 호우 / 태풍 → 풍수해 위기경보 (위기경보 + 기상특보 이중 체계)

**기상특보(호우)**: 주의보 3시간 60mm 또는 12시간 110mm↑ / 경보 3시간 90mm 또는 12시간 180mm↑
**기상특보(태풍)**: 주의보=강풍·풍랑·호우가 주의보 기준 도달 / 경보=그중 하나가 경보 기준 도달 또는 총 강우량 200mm↑

**풍수해 위기경보** (발령자: 행정안전부장관)
| 단계 | 기준 |
|---|---|
| 관심 | 태풍 빈발 시기 / 우리나라에 영향 가능한 태풍 발생 / 호우 빈발 시기 |
| 주의 | 태풍 예비특보 발표 + 대규모 재난 가능성 / 호우 예비특보·주의보 발표 + 대규모 재난 가능성 |
| 경계 | 태풍주의보 발표 + 대규모 재난 가능성 농후 / 호우경보 발표 + 대규모 재난 가능성 농후 |
| 심각 | 태풍경보 발표 + 대규모 재난 발생(가능성 확실) / 호우경보 발표 + 대규모 재난 발생(가능성 확실) |
*출처: 국가위기경보 체계(풍수해 위기경보 표준매뉴얼), 기상청 예보업무규정*

## 홍수 / 침수
- 별도의 자체 위기경보 없음 — **호우·태풍에 의한 「풍수해 위기경보」**에 흡수되어 관리됩니다.
- 하천 수위 기준의 **홍수특보(주의보/경보)**는 국토교통부 홍수통제소가 각 하천 지점별 기준수위로 별도 운영하나, 전국 공통 수치는 지점마다 달라 공개된 단일 기준표는 없습니다.

## 지진
**지진정보 체계**: 지진조기경보=규모 5.0 이상(P파 자동분석) / 지진속보=규모 3.5~5.0 / 지진재난문자=규모 3.0 이상(지역) 또는 5.0 이상(전국)

**지진 위기경보** (발령자: 행정안전부장관)
| 단계 | 기준 |
|---|---|
| 관심 | 규모 4.0 미만(진도 Ⅴ 미만) 지진이 짧은 기간 동안 특정 지역에서 2회 이상 발생 |
| 주의 | 위 조건이 3회 이상 발생 |
| 경계 | 국내 규모 4.0~4.9 지진 또는 최대진도 Ⅴ 발생, 인명·재산 피해(예상) |
| 심각 | 국내 규모 5.0 이상 지진 또는 최대진도 Ⅵ 이상, 피해 확대 예상 |
*출처: 국가위기경보 체계(지진 위기경보 표준매뉴얼)*

| 심각 | 대규모 산사태(발생·확실) 또는 인명피해 / 산사태경보 예측정보 30%↑ 시·군·구 / 지진 규모 6.0 이상 |

우주전파재난은 위 1부의 기준과 동일합니다(과학기술정보통신부·우주항공청 공동 관리). **GPS전파혼신재난**은 이와 유사한 체계로 운영되는 것으로 추정되나, 공개된 4단계 정량 기준은 확인하지 못했습니다.

## 방송통신재난
**방송통신재난 위기경보** (발령자: 과학기술정보통신부장관) — 다른 재난(태풍·호우·지진·전력)의 경보 단계에 연동
| 단계 | 기준 |
|---|---|
| 관심 | 태풍 발생 / 3개 이상 광역단체 호우경보 / 지진·지진해일 "주의" / 한전 전력위기경보 "주의" / 100만 명 이상 행사 트래픽 급증 |
| 주의 | 태풍 직접 상륙 예상 / 5개 이상 광역단체 호우경보 / 지진 "경계" / 전력위기경보 "경계" |
| 경계 | 3개 이상 시·군·구 규모 방송·통신 시설·서비스 피해(자연재해 또는 정전·화재·시스템고장) / 전력위기경보 "심각" |
| 심각 | 권역 규모 방송·통신 시설·서비스 피해 |

- 자연재난 중: 빙판길 낙상사고, 해빙기 재난예방(둘 다 절기 기준 캠페인성 항목으로 단계 자체가 없음), 조수(시기 기준), 자연우주물체추락(관심 단계 실제 발령 사례는 있으나 등급 기준 비공개)
